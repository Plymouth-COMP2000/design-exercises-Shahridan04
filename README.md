# Restaurant Management App (Assessment 2)

## Student ID: BSCS2509260

This is the Android implementation for the Restaurant Management System.

### Third-Party Resources Used
* **Retrofit 2.9.0**: For handling REST API network requests.
* **Gson 2.10.1**: For converting JSON data to Java objects.
* **Room Database 2.6.1**: For local SQLite data persistence.
* **Material Components**: For UI design elements (Cards, Buttons, Inputs).
